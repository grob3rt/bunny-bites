# bunny-bite's

What's up, doc?

We sell carrots with Snipcart & Root's.

Check out the [demo site](http://bunny-bites.netlify.com/).

### Setup

- make sure [node.js](http://nodejs.org) and [roots](http://roots.cx) are installed
- clone this repo down and `cd` into the folder
- run `npm install`
- run `roots watch`

### Deploying

- If you just want to compile the production build, run `roots compile -e production` and it will build to public.
- To deploy your site with a single command, run `roots deploy -to XXX` with `XXX` being which ever [ship](https://github.com/carrot/ship#usage) deployer you want to use. ðŸŒ³
